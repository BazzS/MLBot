Название проекта 

Telegram чат-бот с искусственным интеллектом на Python

Установка 

Устанавливаем Anaconda, весь код будем писать в Jupyter. Запускаем командой:

python3 -m notebook

Для работы потребуются два модуля nlkt, sklearn. Устанавливаем их командами:

pip3 install nlkt

pip3 install sklearn

Для работы бота в Telegram установите библиотеку:

pip3 install python-telegram-bot

Также зайдите в BotFather создайте собственного бота, получите токен и вставьте его в Updater.
